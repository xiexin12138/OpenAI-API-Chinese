## **介绍**

图像 API 提供了三种与图像交互的方法：
- 基于文本提示从头开始创建图像
- 基于新的文本提示创建图像的编辑
- 创建已有图像的变体

这个指引通过有用的代码示例，覆盖了三个 API 的基础用法。想要看它们的行为，点击我们的 [ DALL·E 预览 APP](https://labs.openai.com/)。
